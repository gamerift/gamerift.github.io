Teardown

Release Date: 2022/04/21

Game Description Teardown is a robbery simulator in a variety of conditions prepared by indie developers from Tuxedo Labs. Stylistically, the project was made using a voxel graphic component. This solution made it possible to implement complete destructibility of all available objects in the location. This freedom opens up the possibility of performing operations in a variety of ways, improvising on the fly. The gamer will have at his disposal explosives and a sledgehammer for destroying all kinds of surfaces to penetrate the territory, as well as several types of vehicles of different sizes for loading valuables and other things that need to be stolen according to the terms of a particular task....
