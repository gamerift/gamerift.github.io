Game Description#

Chipwits is a series of logical puzzles built on automation. The player takes the Chipwit CW-03.24 model under the control of the robot and learns to direct his actions on the territory of the workshop. To create sequences of movements and operations, chips with instructions are placed. Skills from past zones are used to solve subsequent, more intricate tests. The main task is to help Professor a chip with the opening of his first factory, having fulfilled all the necessary instructions for each location.

Chipwits is built in such a way as to copy the programming bases, making the project available to beginners. It is necessary to collect command chains, placing the chips in the right order, and then start the algorithm and observe the attempts of the robot to achieve the goal. Each riddle on the floor has several ways to pass. Such a system encourages experiments with various combinations to obtain a result, opening the scope for unique approaches within the framework of the gameplay.

Each solution can be additionally modified to increase efficiency. Gamer can work on maximizing the account or reducing the number of chips involved in the construction of the program. The third optimization path includes a decrease in the number of cycles - steps performed by the program from start to finish. For connoisseurs of accurate measurements, there is a global table of leaders, which monitors and display, as performance indicators correlate with the results of other people.
